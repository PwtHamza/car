Introduction

Welcome to the AutoBxis Car Dealership Website documentation. This guide will help you understand and customize the website according to your needs. AutoBxis is a modern, responsive car dealership website template designed to showcase vehicles, provide detailed information, and facilitate customer inquiries.

Quick Start

To get started with the AutoBxis website:

  1. Upload all files to your web server
  2. Customize the website content in HTML files
  3. Modify styles in CSS files as needed
  4. Update JavaScript functionality if required

Key Features

  • Responsive Design - Mobile-first approach ensuring compatibility across all devices
  • Modern UI - Clean and professional design focused on vehicle showcasing
  • Car & Motorcycle Listings - Dedicated pages for both car and motorcycle inventories
  • Dynamic Components - Carousels, filters, and interactive elements
  • WhatsApp Integration - Floating WhatsApp button for instant customer communication
  • Bootstrap 5 - Built on the latest Bootstrap framework
  • Customizable - Easy to modify with CSS variables and modular components
  • Cross-browser Compatible - Works on all modern browsers

Color Scheme

AutoBxis uses a consistent color scheme defined through CSS variables for easy customization. These variables are defined in the :root selector in the assets/css/styles.css file.

Primary Colors

Variable Color Value Usage
--primary-color #ff4d30 Primary accent color, buttons, links, highlights
--primary-hover #df2a0c Button hover states, active elements
--secondary-color #607d8b Secondary elements, alternative buttons
--white #ffffff Light backgrounds, text on dark backgrounds

Text Colors

Variable Color Value Usage
--text-dark #263238 Main text color, headings
--text-light #546e7a Secondary text, paragraphs
--text-lightest #78909c Tertiary text, metadata

Background Colors

Variable Color Value Usage
--bg-dark #1a1d20 Dark backgrounds, footer
--bg-light #f8f9fa Light backgrounds, alternating sections
--bg-ultra-light #f5f5f5 Very light backgrounds, cards

Customizing Colors

To change the color scheme of the entire website, modify the CSS variables in the :root selector in styles.css file:

:root {
    /* Main colors */
    --primary-color: #ff4d30;  /* Change this to your preferred primary color */
    --primary-hover: #df2a0c;  /* Adjust this to be slightly darker than primary */
    --primary-color-rgb: 255, 77, 48; /* Update these RGB values to match your new primary color */
    --secondary-color: #607d8b;
    --secondary-hover: #455a64;
    --white: #ffffff;
    
    /* Text colors */
    --text-dark: #263238;
    --text-light: #546e7a;
    --text-lightest: #78909c;
    
    /* Background colors */
    --bg-dark: #1a1d20;
    --bg-light: #f8f9fa;
    --bg-ultra-light: #f5f5f5;
    /* ... other variables ... */
}

CSS Naming Conventions

The CSS follows these naming conventions:

  • Component-based classes - Classes are named according to the component they style (e.g., .car-card, .hero-slide)
  • BEM-inspired naming - Elements within components often use a modified BEM approach (e.g., .car-card, .car-card__image)
  • Utility classes - Some utility classes are used for common styling needs (e.g., .text-accent)
  • State classes - Classes that indicate state changes (e.g., .active, .visible)

CSS Variables

CSS variables (custom properties) are used extensively to maintain consistency and make theming easier. Most important variables are declared in the :root selector for global accessibility.

JavaScript

The JavaScript code is distributed across multiple files to maintain separation of concerns and isolated functionality.

Main JavaScript (main.js)

The main.js file handles general website functionality including:

  • Carousel initializations
  • Vehicle filtering
  • Modal functionality
  • Search form handling
  • Vehicle card interactions

WhatsApp Button (whatsapp-button.js)

This isolated script handles the WhatsApp floating button functionality:

  • Dynamically creates the button on all pages
  • Handles button visibility based on scroll position
  • Manages animations and hover effects
  • Sets up the WhatsApp link with the phone number

Scroll to Top Button

The scroll-to-top functionality is implemented directly in the HTML files using an isolated script:

(function() {
    // Isolated scope for scroll to top functionality
    const scrollBtn = document.getElementById('scroll-to-top-btn');
    
    // Show button when user scrolls down 300px
    window.addEventListener('scroll', function() {
        if (window.pageYOffset > 300) {
            scrollBtn.classList.add('visible');
        } else {
            scrollBtn.classList.remove('visible');
        }
    });
    
    // Smooth scroll to top when button is clicked
    scrollBtn.addEventListener('click', function() {
        window.scrollTo({
            top: 0,
            behavior: 'smooth'
        });
    });
})(); // IIFE to avoid global scope pollution

External Libraries

The website uses these external JavaScript libraries:

  • jQuery - For DOM manipulation and other utilities
  • Bootstrap JS - For modals, dropdowns, tabs, and other components
  • Owl Carousel - For the various sliders and carousels
  • Font Awesome - For icons (CSS with JS support)

Adding Custom JavaScript

To add custom JavaScript to the site:

  1. For global functionality, add it to the main.js file
  2. For isolated functionality, create a new JS file and include it in the HTML
  3. Use IIFEs (Immediately Invoked Function Expressions) to avoid polluting the global scope

Customization

The AutoBxis website is designed to be easily customizable. Here are some common customization tasks:

Changing Colors

To change the color scheme, modify the CSS variables in the :root selector in assets/css/styles.css:

:root {
    /* Main colors */
    --primary-color: #ff4d30;  /* Change this to your desired primary color */
    --primary-hover: #df2a0c;  /* Slightly darker than primary */
    /* Other color variables */
}

Changing Logo

To change the logo, locate the logo element in the header section of each HTML file and update it:

<a class="navbar-brand text-white" href="#">
    <h2 class="m-0">AUTOBXIS</h2>
    <!-- Or use an image logo: -->
    <!-- <img src="assets/images/your-logo.png" alt="AutoBxis" height="40"> -->
</a>

Changing Contact Information

To update contact information, modify these elements in the HTML files:

  • Top bar contact links
  • Contact page form and information
  • Footer contact section
  • WhatsApp button phone number in whatsapp-button.js

Adding New Pages

To add a new page:

  1. Copy an existing HTML file that is closest to your needs
  2. Rename the file to your new page name (e.g., new-page.html)
  3. Update the page title and content
  4. Add links to the new page in the navigation menu

Adding New Vehicle Listings

To add new vehicle listings:

  1. Copy an existing vehicle card structure
  2. Update the vehicle information and image
  3. Add it to the appropriate listing section

Modifying the Navigation Menu

To modify the navigation menu, update the <ul class="navbar-nav ms-auto"> element in the header section of each HTML file.

Customizing Forms

Forms use Bootstrap 5 form components. To customize forms, modify the HTML structure and the corresponding CSS styles.
